## Deskripsi Proyek
Proyek ini menganalisis kinerja bisnis Kimia Farma dari 2020 hingga 2023, mencakup penjualan, laba, dan kualitas layanan di seluruh cabang. Data diolah di BigQuery dan divisualisasikan menggunakan Looker Studio.

## Struktur Folder
- `sql/` : Semua query SQL untuk membuat tabel dan view di BigQuery.
- `docs/` : Dokumentasi dataset, schema export, dan instruksi setup.
- `dashboard/` : Screenshot dashboard dan link Looker Studio.
- `terraform/` : (Opsional) Deployment scripts untuk resources GCP.

## Instruksi Setup
1. Buat dataset `kimia_farma` di BigQuery.
2. Jalankan query SQL di folder `sql/` untuk membuat tabel `analisa` dan view `kf_yoy_revenue`.
3. Export schema jika diperlukan (folder `docs/`).
4. Dashboard Looker Studio bisa diakses melalui link di `dashboard/report_links.md`.
